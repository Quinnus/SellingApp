A simple online ordering portal demonstrating polymorphism, 
user input, class Overrides and console output.

Open and run Shopping.java.

User interface will prompt for input, including options to creatie a manual order, 
printing various reports, and exiting the console.

The abstract Order class is used to create a number of different child classes, each of which 
overrides a parent method.

Most classes are abstracted from the main logic.
